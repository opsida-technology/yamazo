# Yamazo VAS

Video Analysis System for IKF Kendo competitions.

## Download

Download the latest version from [Releases](https://github.com/opsida-technology/yamazo/releases).

| Platform | File |
|----------|------|
| macOS (Apple Silicon) | `Yamazo VAS-x.x.x-arm64.dmg` |
| Windows (64-bit) | `Yamazo VAS Setup x.x.x.exe` |

## Installation

**macOS:**
1. Download the DMG file
2. Open and drag the app to Applications folder
3. First launch: Right-click → Open (to bypass Gatekeeper)

**Windows:**
1. Download the EXE file
2. Follow the installation wizard
3. Launch from desktop shortcut
